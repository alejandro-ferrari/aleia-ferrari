# Aleia Ferrari

A Cowork marketplace by Alejandro Ferrari.

## Plugins

- **aleia-cowork** — *Aleia Ferrari for Claude Cowork.* A productivity starter that installs
  a spec-first, evidence-reviewed working method (LonelyOctopus method improved with
  Karpathy's spec-first approach) into Claude Cowork. Currently a v0.0.1 install-validation
  stub.

## Install (Claude Cowork / Claude Code)

1. Add this marketplace: `alejandro-ferrari/aleia-ferrari`
2. Install the **aleia-cowork** plugin.
3. Run its start command in the Cowork chat.
